# 奇门遁甲学习指南

基于《神奇之门》等公开资料整理的奇门遁甲交互式学习网页，包含从基础概念到手动排盘的完整学习路径。

## 功能特性

- **核心概念速览**：阴阳五行、天干地支、九宫八卦的直观图表
- **奇门盘结构**：地盘、天盘、人盘、神盘四层结构的详细说明
- **符号详解**：八门、九星、八神、十干克应的完整参考表
- **节气用局**：二十四节气用局表与拆补法说明
- **排盘起局**：分步教学与示例格局展示
- **实战框架**：各类预测的用神速查与判断思路
- **排盘模拟器**：输入日期时间自动排出时家奇门盘

## 使用方法

直接在浏览器中打开 `index.html`，或通过 GitHub Pages 访问。

### 排盘模拟器

1. 输入公历日期和时间
2. 点击"排盘"按钮
3. 查看自动生成的奇门盘格局（含天、地、人、神四层）
