一、概念入门

奇门遁甲是古代的一种时空数理模型。掌握它需要先理解几个核心概念，它们是搭建整个知识体系的基石。

1.1 阴阳

阴阳阴阳不是「好坏」之分，而是事物的一体两面。白天（阳）与黑夜（阴）、热（阳）与冷（阴）、动（阳）与静（阴）——二者互相依存、互相转化。是古人描述万事万物两种基本状态的符号。它不讲「好坏」，只讲「两面」——就像一枚硬币有正反面。

在奇门遁甲中，阴阳直接决定排盘方向：

  • 阳遁：冬至后阳气渐长 → 顺排六仪、逆排三奇
  • 阴遁：夏至后阴气渐长 → 顺排三奇、逆排六仪

记忆口诀：冬至一阳生→阳遁，夏至一阴生→阴遁。这是所有排盘操作的总开关。

1.2 五行

五行金、木、水、火、土不是五种「物质」，而是五类性质：木代表生长升发，火代表炎热向上，土代表承载化育，金代表收敛肃杀，水代表滋润下行。它们之间的关系是奇门判断吉凶的核心逻辑。描述事物之间的五种基本关系。每种元素代表一类性质，不是具体的物质。

  • 相生（促进）：木生火 → 火生土 → 土生金 → 金生水 → 水生木
  • 相克（制约）：水克火 → 火克金 → 金克木 → 木克土 → 土克水

相生记忆：木燃生火 → 火烧成灰（土）→ 土中炼出金属 → 金属熔成液体（水）→ 水润养草木
相克记忆：木破土而出 → 土筑堤挡水 → 水泼灭火势 → 火熔金属 → 金斧砍断木

1.3 天干地支

天干地支天干地支是中国最古老的计时系统。十天干（甲乙丙丁戊己庚辛壬癸）配十二地支（子丑寅卯辰巳午未申酉戌亥），最小公倍数60，形成六十甲子，用来记录年、月、日、时四个时间单位。奇门遁甲的所有排盘操作，本质上就是在排列组合天干地支。是奇门遁甲的"基本字母"。排盘时处理的一切符号，本质都是天干地支的排列组合。

1.4 九宫八卦

九宫八卦九宫来自洛书——一个神奇的三阶幻方，每行每列每对角线之和都是15。后天八卦来自《周易》，将万物万象归纳为八种符号，并与八方、八节气相对应。奇门遁甲将九宫与八卦结合，构成地盘的固定框架。这个框架是永远不变的。是奇门盘的"棋盘"。九宫来自洛书，八卦来自周易，二者组成了地盘永不移动的框架。

奇门盘像一座四层楼：地盘是地基，天盘和人盘每时辰在转，神盘附在最上面。

2.2 地盘的核心：三奇六仪的排列

地盘上除了固定不动的九宫八卦，还有九个天干在排列——这就是三奇六仪甲隐藏后剩下九个天干：乙丙丁为三奇（三支奇兵），戊己庚辛壬癸为六仪（六支仪仗队）。它们在九宫中的排列顺序永远不变，只是起点和方向在变。。

固定队形：戊 → 己 → 庚 → 辛 → 壬 → 癸 → 丁 → 丙 → 乙 → (回到戊)

这个顺序绝对不变，像一条首尾相连的锁链。放进九宫时：

  • 阳遁：从一宫起顺排（戊入一宫→己入二宫→...）
  • 阴遁：从九宫起逆排（戊入九宫→己入八宫→...）

关键：甲子戊落在几宫，就叫几局。阳遁一局=甲子戊在一宫；阴遁九局=甲子戊在九宫。

2.3 六甲遁藏

"遁甲「遁」是隐藏的意思。「遁甲」就是把元帅甲藏起来。为什么藏？因为甲属木，最怕庚金（阳金克阳木）。所以六甲分别藏在戊己庚辛壬癸六仪之下，让敌人找不到主帅。"就是把甲藏起来：

合称甲子戊甲戌己甲申庚甲午辛甲辰壬甲寅癸
角色元帅大将大将大将大将大将

六甲每人带班10个时辰（一旬），轮流在地盘值班。轮到谁带班，谁所在宫位对应的星和门就是值符和值使值符是天盘上值班的九星，值使是人盘上值班的八门。找到旬首在地盘的宫位→对应该宫的星就是值符、门就是值使。这是排盘第三、四步的关键。。

2.4 天盘和人盘如何转动

1
找旬首：看当前时辰属六甲中哪一甲的"班"（如丁巳时→甲寅旬）
2
找值符值使：旬首在地盘哪个宫 → 该宫对应的星=值符，对应的门=值使
3
转天盘：值符星跟着时辰天干走——时干在地盘哪宫，值符就落哪宫
4
转人盘：值使门跟着时辰地支走——从旬首地支数到当前地支
5
转神盘：小值符永远跟大值符走，落在同一个宫

三、符号详解

排盘之后，每个宫都会同时出现四类符号。理解每个符号的含义，才能读懂盘中的信息。

八门八门代表人事的八种状态和方向。开、休、生为三吉门；死、惊、伤为三凶门；杜门、景门中平。但吉凶不是绝对的——还需要看门与所在宫的五行生克关系：吉门被克则不吉，凶门被克则减凶。代表人活动的八种状态和方位。注意：吉凶不是绝对的，要看门和宫的生克关系。

门宫位方位五行吉凶宜做什么忌做什么
休门休门属水，居坎一宫（正北）。坎宫处冬季最寒冷季节，万物休息冬眠，故为休养生息之地。利于见贵人、谈判、休养，不利行刑断狱。坎一北水吉见贵人、休养、谈判行刑断狱
生门生门属土，居艮八宫（东北）。正当立春之后，万物复苏，阳气回转，土生万物。利于求财（特别是房地产、种植养殖）、出行、嫁娶。不利埋葬治丧。艮八东北土吉求财、房产、种植埋葬丧葬
伤门伤门属木，居震三宫（正东）。震卦主动，动则易伤；又子卯相刑，刑则受伤。为凶门，主疾病刑伤。但适宜讨债、捕盗、渔猎——以暴制暴之事反而可用。震三东木凶讨债、捕盗、渔猎经商、出行
杜门杜门属木，居巽四宫（东南）。与西北开门相对，一开一杜。杜门为藏形之方，适合躲灾避难、保密工作、技术研发。其余事多不宜。巽四东南木中平躲藏、保密、技术大部分吉事
景门景门属火，居离九宫（正南）。烈日当空，大放光明，但也酷暑难耐。宜于文书、献策、宴饮聚会。但要谨防口舌和血光火灾。离九南火中平文书、献策、宴饮求财
死门死门属土，居坤二宫（西南）。与艮宫生门相对，万物春生秋死。为最凶之门，只宜吊丧送葬、刑戮捕猎。其余一切吉事皆不宜。坤二西南土凶送葬、刑戮、捕猎一切吉事
惊门惊门属金，居兑七宫（正西）。正当秋分寒露霜降，金秋肃杀，草木凋零。主惊恐、创伤、官非口舌。适宜诉讼、设疑伏兵、赌博。其余事不宜。兑七西金凶诉讼、设疑、赌博婚嫁、出行
开门开门属金，居乾六宫（西北）。乾为天为父，为八卦之首，万物开端之意。大吉之门，利于开业经商、征战远行、婚娶迁徙、考学参军。乾六西北金吉开业、远行、婚娶—

口诀：吉门克宫吉不就，凶门克宫事更凶；吉门被克吉不就，凶门被克凶不起。

九星九星代表天时——天体运行对人事的影响。天心、天任、天禽、天辅为四吉星；天蓬、天芮、天柱为三凶星；天冲次吉；天英中平。九星的旺衰取决于它对地盘宫的作用（我生之月最旺），而非自身五行。代表天体对人事的影响，也常用来判断人的先天性格。

天芮天芮原名巨门星，属土，居坤二宫。与死门对应，与疾病流行有关，故为病星/凶星。预测疾病必看天芮。天芮落乾兑二宫（旺）则难治；落震巽二宫（受克）则不药而愈。巨门坤二土凶疾病（预测疾病必看）

四、节气与定局

排盘第一步：根据日期确定用阳遁还是阴遁、用几局。这是最关键的一步，定错了局，后面全错。

4.1 定局的逻辑

一年中：每5天（60时辰）排完一种格局。24节气×3元×5天/元=360天（剩下5天多用拆补法处理）。

  • 冬至→芒种（阳气上升）→ 阳遁
  • 夏至→大雪（阴气上升）→ 阴遁

4.4 如何确定上中下元

知道节气后，还需要知道当天属于上元、中元还是下元。方法：找符头符头就是每元第一天的日干支。因为每元第一天必须是甲日或己日，所以符头的天干一定是甲或己。从当天日干支往前找，碰到的第一个甲日或己日就是符头。。

下元辰、戌、丑、未每季最后一个月四季

使用步骤：从当天干支往前找甲日或己日 → 看其地支 → 确定上/中/下元 → 查用局表。

4.5 拆补法

拆补法拆补法是「无闰派」的定局方法。它不需要复杂的置闰计算，原则很简单：（1）节气一到，立刻使用该节气的用局表；（2）仍按日干支符头定上中下元；（3）交节当天元数不满5天为「残局」，用完下元后回头补上。这种方法避免了人为置闰的混乱，更符合自然规律。：不用复杂的置闰，节气一到就换局，元数不够回头补。本模拟器即用此法。

五、排盘起局

掌握前面的所有基础后，就可以动手排盘了。整个过程只有六步。

1

2

4

六、实战预测框架

排好盘后如何判断？核心是选对用神，分析符号之间的生克关系。

6.1 什么是用神

用神用神就是奇门盘中代表你所问事物的那个符号。比如问考试→天辅星；问疾病→天芮星；问求财→生门和戊。选对用神好比看病找对科室——用神错了，整个判断方向就偏了。是奇门盘中代表你所问事物的特定符号。选错用神，后面的分析就全无意义。

6.4 案例解析

案例一：考试预测（《神奇之门》叶梓铨女儿案例）

背景：1997年8月24日卯时（丁丑年戊申月戊戌日乙卯时），叶梓铨女儿在美国参加医生统一考试，成绩优异却被要求解释。其父请杜新会用奇门预测。

排盘：阳遁一局，甲寅旬，值符天蓬星在坎一宫，值使休门在坤二宫。

用神选取：

  • 天辅星（文昌星）= 考生本人
  • 景门（代表考试、试卷）= 考试结果
  • 二者均落离九宫（火旺之地，景门属火，得天时之助）

判断：天辅星得时得地（落离宫火旺之地），景门旺相（与宫同属火），断为成绩优异在前五名。因离宫为九数、景门旺相，断名次很高（后证实为第二或第三名）。

应期：阴历九月（阳历10月7日），离九宫数九，从起局之时算起约一个半月。

实际结果：美国考试中心于1997年10月7日发出证书，10月9日收到。预测完全准确。

经验：考试预测关键看天辅星是否旺相、景门是否得地。天辅星代表考生自身的学识水平，景门代表考试发挥和成绩。

案例二：疾病预测（综合分析）

核心用神：天芮星（病神）——代表疾病的性质、部位和严重程度。

判断框架：

  • 看天芮落宫与日干关系：天芮落宫克日干落宫 → 病情较重；天芮生日干 → 病虽在但易治；日干克天芮 → 病人意志力强能战胜疾病
  • 看天芮所临之门：临死门 → 病情危重；临惊门 → 惊恐引起；临杜门 → 病因不明或阻塞性疾病
  • 看有无医药星救援：乙奇代表药物、天心星代表医生。若乙奇或天心星落宫能够生天芮或克天芮所落之宫，则有救。
  • 天芮旺衰：落乾兑二宫（金旺之地，土生金，天芮受泄）→ 病虽旺但可治；落震巽二宫（木克土）→ 天芮受克，不药而愈；落坤艮二宫（土旺）→ 病缠绵难愈。

疾病定位（根据九宫）：落离九宫 → 心脏/眼睛/小肠；落坎一宫 → 肾/膀胱/泌尿；落震三宫 → 肝胆/足；落兑七宫 → 肺/口舌；落乾六宫 → 头部/骨骼/大肠；落坤二宫 → 脾胃/腹部；落巽四宫 → 胆/神经；落艮八宫 → 脾/关节。

提示：这里的"看病"仅供传统文化研究参考，实际身体不适请及时就医。

案例三：求财预测（综合分析）

核心用神：

  • 日干 = 求测人自己
  • 时干 = 所求之事（财运）
  • 生门 = 利润、收益
  • 戊（天门）= 资本、本钱

判断框架：

  • 看日干与生门/戊的关系：生门生日干 → 利润主动来找你；日干克生门 → 需要辛苦努力才能获利；生门克日干 → 利润难求反而破财
  • 看生门落宫状态：旺相 → 利润丰厚；休囚 → 获利艰难；受克 → 亏损。
  • 看格局：戊落宫出现「青龙返首」（戊+丙）或「飞鸟跌穴」（丙+戊）等吉格 → 财运亨通；出现「青龙折足」（戊+辛）或「贵人入狱」（戊+己）→ 破财之象。

求财方向：生门所在宫位的方位就是有利的求财方向。如生门落艮八宫（东北），就去东北方求财。

经验：求财预测中，戊+丙（青龙返首）格局出现在生门宫或日干宫，是财运强劲的明确信号。反之，戊+辛（青龙折足）出现则要谨慎投资。

案例四：出行预测（综合分析）

核心用神：

  • 日干 = 出行人
  • 景门 = 道路、旅途
  • 九天 = 远行（出国、长途）
  • 开门 = 顺利与否

判断框架：

  • 日干落宫状态：旺相 → 出行人体力充沛、旅途顺利；休囚 → 容易疲劳、遇麻烦
  • 阻隔信号：杜门（阻塞）或死门（危险）临日干或景门 → 不宜出行
  • 凶神信号：白虎临 → 交通事故或意外伤害风险；玄武临 → 盗贼诈骗之患；螣蛇临 → 事情反复、走错路
